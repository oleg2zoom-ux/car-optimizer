# מחשבון החלטת החלפת רכב — v0.7

גרסה זו כוללת את מאגר המחירון v0.6 ואת שינוי הממשק לפי דרישת המשתמש:

1. לפני הכול בוחרים קטגוריה ראשית:
   - מיני
   - משפחתי
   - פנאי שטח

2. בוחרים עדיפות מיון:
   - עלות חודשית מינימלית
   - רכב חדש ככל האפשר במסגרת האילוצים
   - רכב אמין ביותר / סיכון נמוך
   - רכב שמחזיק מחיר בעתיד

3. המודל מתחיל מהרכב הקיים:
   - זיהוי רכב קיים מתוך מאגר רפרנס
   - שווי נוכחי משוער
   - הלוואה קיימת
   - חיסכון עד החלפה
   - מועד החלפה
   - מימון רכב הבא
   - דירוג לפי עדיפות

## קבצי דאטה

- `data/replacement_options.csv` — מאגר אפשרויות רכישה v0.6, 168 שורות.
- `data/vehicle_reference.csv` — רפרנס לרכב הקיים, נגזר ממאגר המחירון.
- `data/price_master_v0_6.xlsx` — קובץ אקסל מלא של המחירון.

## הרצה מקומית

```bash
pip install -r requirements.txt
streamlit run app.py
```

## פריסה ב-Streamlit Cloud

Main file path:

```text
app.py
```

## הערה חשובה

שורות עם `needs_review = כן` הן שורות seed להרחבת המאגר. הן שימושיות לבדיקות, אך לפני החלטה כספית אמיתית צריך לאמת אותן מול מקור רשמי.
