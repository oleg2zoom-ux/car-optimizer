# מחשבון תזמון והחלפת רכב

אפליקציית Streamlit בעברית לבדיקת החלטת החלפת רכב לפי תזמון, שווי הרכב הקיים, הלוואה קיימת, מימון הרכב הבא, ירידת ערך ואילוץ החזר חודשי.

## מהות האפליקציה

האפליקציה בודקת האם עדיף למכור את הרכב הקיים עכשיו, להישאר איתו עוד תקופה ולשלם הלוואה קיימת, למכור בעתיד ולהשתמש בכסף כמקדמה לרכב הבא, לבחור רכב חדש או משומש לפי גיל מקסימלי, ולעמוד באילוץ החזר חודשי של הלוואה או ליסינג.

המודל המרכזי אינו מחשב דלק, ביטוח וטיפולים. אלה נשמרים ללשונית אופציונלית בלבד.

## מה חדש בגרסה 0.3

- נוספה לשונית ייעודית לרכב הקיים.
- נוספה תמיכה בהלוואה קיימת: יתרה, החזר חודשי, חודשים שנותרו וריבית.
- המחשבון מחפש נקודת מינימום בין תזמוני מכירה שונים.
- נוספו אילוצי גיל לרכב הבא: 3, 5, 7, 10, 15 שנים.
- הוחלפו סליידרים בעייתיים בשדות מספריים ובבחירות קבועות.
- עלויות נלוות הופרדו מהמודל הראשי.

## הרצה מקומית

```bash
pip install -r requirements.txt
streamlit run app.py
```

## פריסה ב-Streamlit Cloud

ב-Streamlit יש להגדיר:

```text
Main file path: app.py
```

## הערת נתונים

הקובץ המצורף הוא מדגם הדגמה בלבד. לפני קבלת החלטות רכישה אמיתיות יש להזין נתוני שוק מבוססים: מחירון, מודעות, טרייד-אין, מחיר יבואן ותצפיות מכירה בפועל.
