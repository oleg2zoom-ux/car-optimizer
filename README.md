# מחשבון החלטת החלפת רכב — v0.4

אפליקציית Streamlit בעברית שמתחילה מהרכב הקיים של המשתמש וממליצה האם ומתי להחליף רכב.

## מה חדש ב-v0.4

- זיהוי רכב קיים מתוך מאגר רפרנס.
- הערכת שווי אוטומטית לפי מחיר ידוע, מחיר רכישה או מודל פחת.
- שימוש בתאריך נוכחי אמיתי לחישוב גיל רכב.
- קנס סיכון גיל/אחריות:
  - בתוך אחריות — סיכון נמוך.
  - אחרי אחריות ועד 7 שנים — סיכון עולה.
  - 7–10 שנים — סיכון בינוני.
  - 10–15 שנים — סיכון גבוה.
  - מעל 15 שנים — סיכון גבוה מאוד.
- התחשבות בהלוואה קיימת.
- המלצה להמתין, לסיים הלוואה ולהמשיך לחסוך אם זה עדיף.
- בדיקה לפי אילוץ החזר חודשי.
- סינון רכב הבא לפי חדש/משומש וגיל משומש מקסימלי: 3, 5, 7, 10, 15 שנים.

## הפעלה מקומית

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Streamlit Cloud

יש להעלות את כל התוכן ל-GitHub כך ש-`app.py` יהיה בשורש ה-repository.

Main file path:

```text
app.py
```
